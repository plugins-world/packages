## 安装

你可以通过 composer 安装这个扩展包，与应用插件不同的是，此扩展会安装到 `vendor/` 目录下

```bash
composer require plugins-world/market-manager
```

## 使用

访问路由：`/market-manager`

注意事项：项目需要配置好权限，避免 web 程序的用户无法读取、创建目录。

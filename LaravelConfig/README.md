# LaravelConfig

一个 Laravel 的全局配置扩展包。

项目自动拆分，如需跟踪源码更新情况，请前往：https://github.com/plugins-world/packages 查看 LaravelConfig 目录

## 安装

```shell
$ composer require plugins-world/laravel-config -vvv
```

## 使用

**规则说明**

```
Model 模型定义
Model\Traits\ServiceTrait 模型操作函数
Helpers\ 数据处理查询操作函数
Utilities\ 增删改便捷操作函数
```
